更新時間：2026-03-26 11:00 CST

## 一、資料來源

### A) Claude MCP（Anthropic Registry）
- 來源：`https://api.anthropic.com/mcp-registry/v0/servers`
- 取法：抓商業可見 servers，僅保留能對應到 GitHub repo 的項目，再查該 repo stars。

### B) GitHub MCP
- 來源：GitHub Search API
- 條件：`mcp-server in:name pushed:>=最近7天`，依 `stars` 由高到低。

> 說明：GitHub API 沒有直接提供「本週新增星數」欄位；這裡用「最近 7 天有更新 + 目前總 stars」作為本週熱門代理指標。
