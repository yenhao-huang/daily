# 熱門 MCP Server 與 Skill 整理

更新時間：2026-03-26 10:28:26 CST

## 一、熱門 MCP Server（實戰常見）

### 1) 官方基礎參考（Core）
1. Filesystem MCP  
   - 讓模型在白名單目錄進行安全檔案操作（讀寫/整理）。
2. Fetch MCP  
   - 抓取網頁內容並轉為 LLM 易讀格式。
3. Git MCP  
   - 查詢/操作 Git repo、diff、提交流程輔助。
4. Memory MCP  
   - 提供持久記憶與知識關聯能力。
5. Sequential Thinking MCP  
   - 結構化思考與分步推理。

### 2) 開發者最常用整合
1. GitHub MCP  
   - Issue/PR/Repo 操作、CI 查詢。
2. Postgres / SQLite MCP  
   - DB 查詢與結構檢視（建議唯讀）。
3. Slack MCP  
   - 查訊息與工作流整合。
4. Brave Search MCP  
   - 即時網路搜尋能力。
5. Browser Automation 類 MCP（如 Puppeteer/Playwright 生態）  
   - 自動化網頁流程、抓取與測試。

### 3) 企業常見整合方向
1. Google Drive / Workspace 類  
2. Sentry 監控與錯誤追蹤  
3. 雲平台與資料平台 MCP（如 Aiven / Algolia / 雲端 DB）

## 二、熱門 Skill 類型（工作流模板）

### 1) Daily Ops
- 每日/每週排程產報、寫檔、commit + push。

### 2) Research Digest
- 多來源抓取、容錯、摘要與趨勢整理。

### 3) Coding / Debug Pipeline
- 任務拆解、程式修改、測試與回報。

### 4) Knowledge Sync
- 與 Notion/文件系統同步與重構內容。

### 5) Infra Healthcheck
- 主機安全健檢、更新策略與風險評估。

## 四、快速落地建議

若要先求穩定與高產出，建議優先組合：
- Filesystem + Fetch + Git + GitHub + Postgres(唯讀)

安全原則：
- 最小權限、白名單目錄、DB 唯讀、對外發送動作加人工確認。
