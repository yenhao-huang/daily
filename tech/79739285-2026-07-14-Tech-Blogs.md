### Reducing High-Bandwidth Memory Bottlenecks in JAX-Based LLM Training with Host Offloading
- 日期：2026-07-10
- 摘要：文章討論 JAX LLM 訓練中 GPU HBM 記憶體瓶頸，尤其模型權重、梯度與 optimizer state 先耗盡記憶體而非算力。解法是透過 host offloading 把部分狀態移出 GPU，以換取更大模型或 batch 的訓練可行性。
