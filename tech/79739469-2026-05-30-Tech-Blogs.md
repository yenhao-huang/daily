### How to Automate AI Model Documentation with the NVIDIA MCG Toolkit
- 日期：2026-05-29
- 摘要：NVIDIA 介紹 Model Card Generator（MCG）Toolkit，可從 GitHub、GitLab、Hugging Face、PDF、DOCX、Markdown 等來源自動擷取資訊，生成 Model Card++ 格式的模型文件。流程採 Ingestion → Extraction → Rendering 架構，結合 Nemotron RAG、NIM、嵌入與 reranking，並支援 Bias、Explainability、Privacy、Safety & Security 子卡片。此工具主打法規合規與稽核可追溯性，能在多數模型倉庫於約 1 分鐘內產出完整模型卡。
