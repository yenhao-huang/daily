# 2026-06-03 AI 新聞

1. **微軟、Google 加速搶 AI 程式開發入口**
   微軟與 Google 近期把 AI coding tools 視為追趕 Anthropic、OpenAI 的關鍵戰場，市場焦點轉向開發者工具與企業導入。這代表 AI 競爭正從聊天機器人延伸到軟體開發流程本身。
   來源：https://www.cnbc.com/2026/06/01/microsoft-and-google-take-on-anthropic-and-openai-in-ai-coding-models.html

2. **Claude Code 領跑，AI coding 市場競爭升溫**
   鉅亨報導指出 Claude Code 帶動代理式寫程式工具熱度，微軟與 Google 也加快反攻，開發者入口成為新一輪 AI 平台戰核心。企業客戶對程式生產力與安全治理需求同步升高。
   來源：https://news.cnyes.com/

3. **Anthropic、OpenAI IPO 與估值話題受關注**
   市場持續關注 Anthropic 與 OpenAI 是否尋求公開市場定價權，AI 基礎模型公司的融資、營收與企業客戶成長成為投資人觀察重點。高估值也讓外界更關注獲利路徑與資本支出壓力。
   來源：https://buzzorange.com/techorange/

4. **AI 原創內容與新聞授權爭議延燒**
   紐約時報發行人示警，AI 若未妥善處理原創報導與授權，可能傷害公眾取得可靠資訊的基礎。媒體業與 AI 公司之間的資料使用、摘要呈現與補償機制仍是監管焦點。
   來源：https://news.ltn.com.tw/

5. **代理式安全工具進入 AI 編碼助手場景**
   Salt Security 推出 Salt Code，主打在 AI 編碼助手內部強制執行安全政策，反映企業對 AI 產生程式碼的合規與漏洞風險更敏感。AI coding 普及後，安全檢查正往開發流程前段移動。
   來源：https://www.moomoo.com/

6. **Palantir 等 AI 概念股持續受資金追捧**
   AI 軟體與資料分析公司仍是市場關注焦點，Palantir 盤中走強凸顯投資人對 AI 應用層營收成長的期待。不過估值偏高也使短線波動風險升高。
   來源：https://www.tradingkey.com/
