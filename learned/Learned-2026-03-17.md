## 今日學到
- 當 `web_search` 缺少 Brave API key 時，改用可取得資料來源（如 RSS + `web_fetch`）可維持日報產出。
- 多代理系統設計上，先明確切分角色（規劃、研究、實作、審核）再分派任務，能降低協作混亂。
